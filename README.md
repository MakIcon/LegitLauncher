# LegitLauncher

Минималистичный Minecraft лаунчер на WPF (.NET)  
Поддержка Vanilla, Fabric и модов с Modrinth.

---

## Возможности

- Установка версий Minecraft (release / snapshot / pre-release)
- Установка Fabric (loader + game version)
- Запуск с выбором:
  - ника
  - объёма RAM
  - пути к Java
- Поиск и установка модов с Modrinth
- Удаление модов
- Отображение использования памяти
- Автоопределение Java (8 / 17 / 21)

---

## Требования

- Windows 10/11
- .NET SDK
- Java 8 / 17 / 21
- Интернет для скачивания версий и модов
